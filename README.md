<h1>DevSorteio</h1>
<br>
<h2>Este é um sorteador de números que aprendi no <a href="https://rodolfomori.com.br/devclub">DevClub</a></h2>
<br>
<p>Bom, este site é um sorteador de numeros, usando o Math.random() para fazer o sorteio. É simples porem funcional, o valor sorteado aparece em um alert na tela do usúario.</p>
